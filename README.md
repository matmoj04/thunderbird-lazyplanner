# Thunderbird Lazyplanner

Small extention to Thunderbird which allows user to track, in my case the progress of the semester in span of 14 weaks. I was fed up by loading Libre Calc to track my ToDo's while
actively using TB and simultaneously finding the Tasks function unsuitable for my use.

## Use
- Create a table.
- Enable Edit mode of the table.
- Set the starting date of the semester.
- Add a new row and rename the subject.
- Add abbriviation to the subject for better readability.
- You can grab and move, delete tabs while in Edit mode.
- Turn off Edit mode.
- By left clicking the cell you cycle through icons.
- By right clicking the cell you add a comment to the cell.
